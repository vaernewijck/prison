Prison API

Available Endpoints

  • /simple.json - Only episode titles
  • /advanced.json - All episode info

How to Use

Access any endpoint directly via URL:

https://vaernewijck.github.io/prison/api/movies.json

Example: Fetch movies with JavaScript


const BASE_URL = 'https://vaernewijck.github.io/prison/api';
          
const fetchData = async () => {
  try {
    const response = await fetch(`${BASE_URL}/advanced.json`);
    const data = await response.json();
    console.log(data);
  } catch (error) {
    console.error('Error:', error);
  }
};

fetchData();
